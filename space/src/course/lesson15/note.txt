代码重用
    是什么？
        父类的代码，子类直接有

    还有哪个语法能做到这一点？
        组合

    继承 vs 组合
        class A extends B  {

        }

        class A {
            B b;
        }

    能用组合 就别用继承

开放类型/不确定类型 效果
    是什么？
        一个父类型变量 能接收任何 子类型的对象

    还有哪个语法能做到这一点？
        接口

多态效果
    是什么？
        调用一个父类型的方法，实际执行 可能 会进到不同的子类的方法

    还有哪个语法能做到这一点？
        接口
